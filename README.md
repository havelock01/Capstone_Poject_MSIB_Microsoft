# Prediksi Harga Bangunan Untuk Kebutuhan Bisnis Real Estate
MariBisnis ingin mengetahui Harga bangunan untuk kebutuhan bisnis Real Estate. Bisnis Real Estate ini merupakan bisnis properti bangunan, dimana bila bisa membeli bangunan yang dilelang dengan harga murah dapat diolah menjadi bisnis yang menguntungkan.
MariBisnis ingin data yang telah didapat dilakukan training dengan Azure Machine Learning dan dibuat laporan dashboard di aplikasi Power BI.

# Tahapan Proses
1. Membuat azure machine learning resource
2. Buat compute instances
3. Buat compute cluster
4. Buat compute inference
5. Buat dataset, upload file dari local
6. Design Model pipeline
7. Publis Inference Pipeline dan Deploy
8. Ambil data csv dan impor ke Power BI
